# Projet_Climat
Ce projet est de regarder l'évolution de :
- la température,
- du CO2,
- de l'agriculture,
- l'accès à l'eau
- PIB
Des années 1960 à 2020.
Et faire des prédictions pour voir dans quel sens irons-nous à l'avenir.
